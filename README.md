# AZ-104-MicrosoftAzureAdministrador-Laboratorio-05---Implementar-conectividad-entre-sitios
En este laboratorio explora la comunicación entre redes virtuales. Implementas peering de red virtual y pruebas conexiones. También crema una ruta personalizada.

Habilidades laborales
Tarea 1: Crear una mañana virtual en una roja virtual.
Tarea 2: Crear una mañana virtual en una roja virtual diferente.
Tarea 3: Utilice Network Watcher para probar la conexión entre máquinas virtuales.
Tarea 4: Configurar pares de redes virtuales entre diferentes redes virtuales.
